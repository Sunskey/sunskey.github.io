《做时间的朋友读书笔记》

发表于 2021-01-10 分类于 读书 评论次数：

作者：李笑来

著作：《把时间当朋友》、《21天TOEFL核心词汇21天突破》、《财富自由之路》、《韭菜的自我修养》

新东方高级教师、创建比特基金专注互联网、比特币的投资。

何为心智

时间是不可以管理的，时间按不按人的意志一步一步的按照固定的频率流逝，唯独可以管理的是我们自己。

一个人的心智就是其过往获得的一切知识以及经验总和（包括基于这些知识和经验造就的思考方法、思考模式）。

想要速成的原因

1.自己的欲望无限

2.要完成的事情太多

3.自我满意度不断的下降

与时间做朋友

每一时刻的现状都是过去某一时刻或者某些时刻的结果，而每一个时刻的现状都是未来某一或者某些时刻的原因。

用正确的方法做正确的事。

侯世达法则：完成一个任务实际花费的时间总会超过计划花费的时间，就算制定计划的时候考虑到本法则，也不能避免这种情况的发生。

为什么人不及时行动

内因：害怕做不好，害怕失败。

外因：过分在意外人的评价。

关注步骤：内容（what）、原因（why）、方法（How）

何为成功

就是获得预期的结果，达到目的。

学习

学习是投资回报最高的行为。

获取知识的主要途径

1.体验、2.试错、3.观察、4.阅读、5.思考

思考

独立的思考、只不过是从别人那里知道一个结论的时候，自己动脑重新推演一遍，看看得出结论的过程有没有漏洞和不合理的地方，衡量一下结论到底有没有道理。

安慰剂效应：1955年，亨利·诺尔斯·比彻在《行之有效的安慰剂》一文中分析了1082个患者的15个临床试验，得出结论：平均35.2%±2.2%的治疗效果来自慰安剂。

自证预言：如果人们相信某件事情会发生，那么这件事情最终真的会发生。

对立论证

人们可能基于一模一样的原因做出截然相反的决定。

钱钟书先生有一段很有趣的描述：“一个人的缺点正像猴子的尾巴，猴子蹲在地面的时候，尾巴是看不见的，知直到他向树上爬，就把后部供大家瞻仰，可是这红臀尾巴本来就有，并非地位爬高了 的新的标志”。

不做事的话，永远不会暴露缺点。

成功的人为啥往往会夸大其词原因

1.获得满足感。

2.保护自己的利益。

3.来自的观众的支持。

自利性偏差：把成功归于自己身上，把失败归咎于别人。

即便是一只坏掉的表，一天之中起码还能对两次呢。

交流

交流的有效往往源自听着而非说者，失败的交流往往源自听者的疏忽。

有效倾听的步骤

1.克制过早的质疑

2.“回顾”记着那些重要点

3.“预期”预判说者接下来的说辞，可以自动集中听者的注意力。

说与不说

通过努力、通过积累成为能说更多话的人。

苏洵：“知无不言，言无不尽，百人誉之不加密，百人毁之不加疏”

《论语》:”可言而不言，失人，不可言而言，失言”

交流守则

1.双方不仅要相互竞争，更要相互合作（秀才遇到兵，可是兵有武器）

2.真理是独立存在的，不会依附个体或者集体

3.真理不变，也不会因任何人而变；不变变化的只有人们对真理的理解。

沟通的本质

沟通本质上来讲是编码和解码的过程，为了保证沟通顺利，往往需要添加一个验证机制。

兴趣

往往不是有兴趣才能做好了，而是做好了才有兴趣。

所有成功的学习，都只依靠两件事策略和坚持，而坚持本身就是最重要的策略。

”’遗忘痛苦”是大脑的自我保护功能，但是这种“善意”的功能也同样有副作用。

运气

一个人觉得“幸运”，是因为他的欲望得到了满足；而一个人觉得“不幸”，是因为他的欲望遭遇了落空。

相信运气其实是缺乏自制力的表现。

苏轼：“卒然临之而不惊，无故加之而不怒”。

所谓幸运就是当你准备好的时候机会就来了，前提是准备好。

人脉

所谓的人脉只不过是某种意义上的“利益交换”。

集中精力改变那些自己能够改变的事情，暂时忽略那些自己不能改变的事情。专心打造自己，让自己成为一个优秀的人、一个有用的人、一个独立的人。

感觉

相对而言，”自卑”、“自信”、“自负”都是感觉的产物，感觉本身就是不精准的，是相对的。

灵感

有目的的积累，灵感来自积累。

李敖：“妓女不需要靠性欲来接客，作家不应该靠灵感来写作”

效率

任何人都不可能100%地有效率，至少不可能总是100%地有效率。

不妨利用黄金分割法。娱乐和学习的比例为0.618。

积累

要相信积累的力量，如果感觉积累已经来不及了，那就接受现实吧！想想可以改变什么、什么不可以改变、找可以改变的地方。

俞敏洪：能够爬上金字塔顶端的动物。雄鹰和蜗牛。蜗牛爬上顶端可能需要一辈子的努力，但是他所达到的高度和看到的世界一样，我们大部分人都不是雄鹰，但是可以不断地攀登自己生命的高峰，终有一天，我们可以在无限风光的险峰俯视和欣赏这个世界。

战国时期的第一个霸主-魏国

发表于 2020-12-30 更新于 2020-12-31 分类于 历史 评论次数：

1.李悝变法

李悝变法的主要内容

李悝变法，是战国初年魏文候当政时，于公元前422年，任命李悝（前455年－前395年）为相，实行变法。

变法的内容主要包含以下4个部分。

  • 废除世袭制度，根据才能选择官吏，招纳贤才。
  • 废除井田制，鼓励老百姓进行垦荒，估算合理的国家土地产量，指定合理的税收政策。
  • 建立了《法经》，针对官员的奖励和惩恶，国家职能等做了完备的规定，基本的国家的法令。
  • 改革军事制度，建立“魏武卒”，免除其家的赋田税。

李悝变法的意义

1.剥削了旧贵族的势力，使得普通老百姓也能够通过才能去晋升。

2.使用包产到户的政策，极大的提高了农民的积极性。同时，根据合理的税收政策和估算，计算出国家的国力，为国家稳定的发展提供了一定的根基。

3.以法治国，使得各种行为和准则都有了指导的方针。《法经》分为《盗法》、《贼法》、《囚法》、《捕法》、《杂法》、《具法》6篇，后续的秦朝的法律也在此基础上建立而成。具有划时代的意义。

4.极大提高了士兵们的积极性，建立了一只强大的军队。

2.魏国强大的根本

  • 政治优势

1.人才济济。魏文侯很会识人，他手下的班子吴起、李悝、任座、魏成、乐羊、西门豹等。

2.打通了人才经络，废除了旧的世袭制度，开始根据能力选拔官员。

3.打通了土地经络，极大的提高了农民的积极性，帮助魏文侯摸清了国家的实力。

4.出台《法经》，使整个国家运行在法制之下。

5.打通军事经络，建立“魏武卒”，奖励优秀者。

  • 军事优势

第一个实行军事改革的国家，拥有强大的“魏武卒”，同时拥有战国时期初年第一站神吴起，在吴起的带领下，进行了72场战斗，取得了64胜，8平的优秀战绩。

  • 经济优势

魏国的国土拥有丰富的资源：小米和盐。小米一方面能够提供丰富的营养价值，另一方面，拥有优秀的保质期。最长的存储时间可以达到9年，是大米的两倍。其次，魏国的洼盐补充了人们对盐的需求，盐也可以作为腌料，用来储存猪肉等食品。

  • 地理优势

魏国的国土总共分为四块，河内、河东、河西、河间之地。

河东（今山西运城）：丰富的盐和铁矿资源，东连上党，西面黄河，北隔晋阳，南面直通中原，进可攻，退可守。魏国的政治和文化中心。

河西（今陕西延安）：虽然穷，但是享受黄河之限。

河内（今焦作、新乡、安阳等地）：人民的文明程度比较高，地址险恶，与其他其他国家的土壤详接，退无可守，属于四站之地。

河间：拥有两道在后面一千年内都被称为华夏最重要的地理通道：豫西和豫北通道。豫北通道是东西跨越太行山脉的最省力途径。豫西通道则是黄河以南连接东西的最主要的交通线。而豫西通道中有着战略地位极其重要的“函谷关”。“一夫当关万夫莫敌”说的就是函谷关，易守难攻，同时可借助黄河之力，运送物资。

河内的扩展成本极低：

  • 土地肥沃，粮食产量高。
  • 地址平坦，便于部队投放。
  • 周边国家力量不强，用兵的损失小，收益大。
  • 中华名族的发祥地，政治上的意义比较高。

3.桂陵之战、马陵之战

  • 桂陵之战

得到韩昭侯、秦孝公、齐威王的不帮架的承诺后，兴兵伐赵，直逼邯郸，齐国派出孙膑营救赵国，历史上著名的”围魏救赵“。 孙膑把庞涓引到齐国的土地后，利用“不知战之地，不知战之时”的主场优势，大败魏军。打破了魏国不可战胜的神话。

  • 马陵之战

桂陵之后的十三年，魏国伐韩，想一举灭掉韩国，齐国本着“中原均势”的政治战略，派出孙膑解救韩国，还是一样的配方，一样的味道，指至魏国国都大梁，魏惠王举全国之力，转而进攻齐国，孙膑再次引诱庞涓，采用“减灶法”，让庞涓以为齐国人都怂了，逃跑了。引庞涓来到马陵（山东濮县），利用地理优势，再次大败魏军。从此，魏国一蹶不振，国力大大衰减。

《孙子兵法》：“主不可怒而兴师，将不可愠而致战”。

马陵之战是秦国与中原大地的重大的历史转折点。

4.衰落的原因

战略的错误：错误的选择战场，一直把中原之地作为首要目标。如果选择攻打秦国，坐拥函谷关和黄河之险，养精蓄锐，自东向西的一步一步的攻打其他国家，那么最后历史的走向很有可能不同。

线程安全学习笔记(一)

发表于 2020-12-25 分类于 后端 评论次数：

线程安全

线程安全是一个多线程环境下正确性的概念，也就是保证多线程环境下、可修改的状态的正确性。

线程安全基本特性

  • 原子性，简单来说相关操作不会被其他线程干扰，一般通过同步机制实现。
  • 可见性，一个线程修改了某个共享变量，其状态能够立即被其他线程知晓，volatile保证可见性。
  • 有序性，保证线程内串行语义，避免指令重排。

所谓的公平性是指在竞争场景中，当公平性为真时，会倾向于将锁赋予等待时间最久的线程 。公平性是减少线程“饥饿”。

如果说ReentranLock是synchronized的替代选择，Condition则是将wait、notify、nofityAll等操作转为为相关的对象，将复杂而晦涩的同步操作转变为直观可控的对象行为。

Condition通过signal/wait的组合，完成了条件判断和通知等待线程。

Sync和Lock的区别

1.用法比较

Lock使用起来比较灵活，但是必须有释放锁的配合动作。

Lock必须手动获取和释放锁，而synchronized不需要手动释放。

Lock只适用于代码块锁，而synchronized可用于修饰方法、代码块等。

2.特性比较

能被中断的特性：与synchronized不同，获取锁的线程能够响应中断，当获取到锁的线程被中断时，中断异常将会被抛出，同时锁会被释放。

3.注意事项

一定要在finally中，释放锁，ReentranLock提供的newCondition的方法，以便用户在用一锁的情况下可以根据不同的情况执行等待或者唤醒的动作。

Synchronized实现

sync代码块是由monitorenter/monitorexit 指令实现的，Monitor对象是同步的基本实现。

Java提供了三种被不同的Monitor实现，三种不同的锁：偏向锁、轻量级锁和重量级锁。

所谓锁的升级、降级、就是JVM优化synchronized 运行的机制，当JVM检测到不同的竞争状况时，会自动切换到合适的锁实现。

当没有竞争时，默认会使用偏向锁。JVM会利用CAS操作，在对象头上的Mark Word 部分设置线程ID，如果有另外的线程试图锁定某个这个已经被偏斜的对象，JVM就需要撤销偏向锁，并切换到轻量锁的实现。轻量级锁通过CAS操作Mark Word来试图获取锁，如果重试成功，就使用普通的轻量级锁，否则，进一步升级为重量级锁。
